1# elliottluftman.github.io

Personal portfolio website — dark terminal / cybersecurity aesthetic.

## Stack

Pure HTML · CSS · Vanilla JavaScript — no frameworks, no build step.

## Setup

1. Clone the repo and add your images to `images/`:
   - `headshot.jpg`
   - `usc_logo.png`
   - `cybernet_logo.png`
   - `resonant_logo.png`
   - `nasa_logo.png`

2. Fill in the three placeholder project cards in `index.html` (search for `TODO:`).

3. Update the GitHub social link in the Contact section with your GitHub profile URL.

4. Push to `main` — GitHub Pages will serve `index.html` automatically.

## Structure

```
├── index.html       # Single-page site
├── style.css        # All styles (CSS custom properties at top)
├── script.js        # Typing animation, tabs, scroll effects
└── images/          # Add your images here
```
